Proyecto Feria Itinerante: Conciencia Verde


Introducción

El objetivo de este sitio web es presentar una feria itinerante que promueve la sustentabilidad y el consumo responsable, dirigida a todo tipo de públicos, desde familias hasta jóvenes interesados en productos ecológicos.


Descripción del Proyecto

La feria recorrerá diferentes ciudades de una provincia argentina, ofreciendo actividades educativas y recreativas sobre prácticas sustentables. El sitio web servirá como plataforma para informar sobre la feria, sus actividades y productos disponibles.


Identidad Visual

El diseño del sitio web se basa en un concepto natural y ecológico, utilizando colores cálidos y tipografías legibles para transmitir los valores de la sustentabilidad.
